# CV Studio v6

Un éditeur de CV professionnel entièrement dans le navigateur — sans installation, sans compte, sans données envoyées à un serveur.

---

## ✨ Fonctionnalités

- **40+ mises en page** — Terminal, Banner, Nordic, Swiss, Neon, Infographic, Scientific, Twin, et bien d'autres
- **Palettes de couleurs** — Choix parmi plusieurs thèmes d'accent (Emerald, Indigo, Rose, Amber…)
- **Typographie personnalisable** — Sélection de la police de titre et du corps, taille, interlignage, espacement
- **Photo de profil** — Importation depuis votre appareil, affichée en cercle sur le CV
- **Compétences avec barres de progression** — Niveau en pourcentage, visuel automatique
- **Sections activables/désactivables** — Profil, Expériences, Projets, Compétences, Outils, Langues, Formations, Certifications
- **Densité et zoom** — Ajustement de l'espacement et de l'échelle d'affichage
- **Mode clair / sombre** — Bascule instantanée pour l'interface
- **Sauvegarde automatique** — Données conservées dans le navigateur (localStorage)
- **Export multiple** :
  - PDF (impression directe depuis le navigateur)
  - PNG / JPEG (capture haute résolution ×2)
  - Word (.doc, compatible Microsoft Office)
  - JSON (sauvegarde et restauration complète du CV)

---

## 🚀 Utilisation

1. Ouvrez le fichier `index.html` dans votre navigateur (Chrome ou Edge recommandé)
2. Remplissez vos informations dans le panneau de gauche
3. Choisissez une mise en page dans le rail de droite
4. Personnalisez couleurs et typographie via les paramètres
5. Exportez votre CV au format de votre choix

> Aucune connexion internet requise pour utiliser l'éditeur (hors chargement des polices Google Fonts).
